Гипсокартонный лист
Гипсокартонный лист - отделочный материал, представляющий собой идеально ровный лист, скомпонованный из двух пластов облицовочного картона. Между слоями картона находится сердечник из твёрдого гипса. С помощью него можно легко выровнять даже самую неровную стену и потолок при внутренней отделке помещений. Также ГКЛ активно используется для возведения перегородок разной конфигурации, обшивок и для монтажа различных элементов внутреннего декора в сухих отапливаемых зданиях.
